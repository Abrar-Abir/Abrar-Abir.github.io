README — 3D Structure Generator

3D Structure Generator

Interactive browser-based tool for visualizing geometric 3D structures as ball-and-stick models. No installation required — open index.html in any modern browser.

Usage

  1. Type a shape name or click a preset
  2. Press Enter or click Generate
  3. Adjust sliders and colors live

Navigation: drag to rotate  ·  scroll to zoom  ·  right-drag to pan

Supported Shapes

Category Examples
Platonic solids tetrahedron, cube, octahedron, icosahedron, dodecahedron
Pyramids square pyramid, pentagonal pyramid, any n
Bipyramids triangular bipyramid, pentagonal bipyramid
Prisms triangular prism, hexagonal prism
Antiprisms square antiprism, pentagonal antiprism
Flat polygons pentagon, hexagon, 8-gon
Special benzene, dimer

The parser is flexible — 5-pyramid, pentagonal-prism, and 8 antiprism all work. Any n from 2–20 is accepted.

Parameters

Control Effect
Bond Length Scales all edges uniformly
Apex Height Factor Stretches/squashes pyramids and bipyramids
Prism Height Factor Stretches/squashes prisms and antiprisms
Antiprism Twist Rotates top ring (0° = prism, 45° = regular antiprism)
Atom / Bond Radius Size of spheres and cylinder thickness
Colors Atom color, bond color, background color
