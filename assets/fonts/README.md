# Departure Mono

Download `DepartureMono-Regular.woff2` from https://departuremono.com/
and place it in this directory as:

    assets/fonts/DepartureMono-Regular.woff2

The site's `@font-face` declaration (in `_sass/_departure.scss`) points to
`/assets/fonts/DepartureMono-Regular.woff2`.

Also drop the font's OFL license file here as `LICENSE.txt`.
