README — 3D Structure Generator

3D Structure Generator

Interactive browser-based tool for visualizing geometric 3D structures as ball-and-stick models. No installation required — open index.html in any modern browser.

Usage

  1. Type a shape name or click a preset
  2. Press Enter or click Generate
  3. Adjust sliders and colors live

Navigation: drag to rotate  ·  scroll to zoom  ·  right-drag to pan

Supported Shapes

CategoryExamples
Platonic solidstetrahedron, cube, octahedron, icosahedron, dodecahedron
Pyramidssquare pyramid, pentagonal pyramid, any n
Bipyramidstriangular bipyramid, pentagonal bipyramid
Prismstriangular prism, hexagonal prism
Antiprismssquare antiprism, pentagonal antiprism
Flat polygonspentagon, hexagon, 8-gon
Specialbenzene, dimer

The parser is flexible — 5-pyramid, pentagonal-prism, and 8 antiprism all work. Any n from 2–20 is accepted.

Parameters

ControlEffect
Bond LengthScales all edges uniformly
Apex Height FactorStretches/squashes pyramids and bipyramids
Prism Height FactorStretches/squashes prisms and antiprisms
Antiprism TwistRotates top ring (0° = prism, 45° = regular antiprism)
Atom / Bond RadiusSize of spheres and cylinder thickness
ColorsAtom color, bond color, background color
